# 🏀 NBA Simulação em Java
Este é um projeto simples desenvolvido em **Java**, inspirado em jogos de basquete como *NBA2K*.  
O objetivo principal é praticar **lógica de programação** e **orientação a objetos**, criando uma simulação básica de personagens e evolução de atributos.
---
## 🎯 Funcionalidades atuais
- Criar personagem com atributos iniciais.
- Treinar personagem:
  - +2 pontos de Overall por treino.
  - Se o personagem estiver com 98 de Overall, o treino adiciona apenas +1 (máximo 99).
  - Se o personagem já estiver com 99, aparece mensagem informando que atingiu o nível máximo.
- Mostrar dados do personagem.
- Mensagens de erro quando:
  - Tentar mostrar dados sem personagem criado.
  - Tentar treinar sem personagem criado.

=======================================================

Estrutura do Projeto:

- Classe Jogador: contém atributos como nome, idade, altura, posição dentro de quadra, college que frequentou e overall, além dos métodos de treino e exibição de dados.

- Classe Main: responsável pelo fluxo principal e interação com o usuário.
Métodos principais:

- criarJogador()

- treinarJogador()

- mostrarDados()
